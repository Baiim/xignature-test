# xignature

Xignature is Project for see Post and upload PDF

_Technology_

```
Front-End : VUE js + Tailwind CSS + AXIOS + VUEX
```

```
Backend : NODE Js + express
```

```
Database : MongoDB
```

## Frontend Setup

## Project setup

```
npm install
```

### Run Project

```
npm run serve
```

## Backend Setup

- on terminal switch to folder backend with command

```
cd backend
```

## Project setup

```
npm install
```

### Run Project

```
npm run dev
```

## MongoDb

## Start your mongoDb server

```
~/mongodb/bin/mongod
```

```
~/mongodb/bin/mongo
```
